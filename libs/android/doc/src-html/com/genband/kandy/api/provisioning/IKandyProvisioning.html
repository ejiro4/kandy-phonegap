007 * <p>
008 * <b><i>Subscriber Provisioning</i></b>
009 * <p>
010 * This interface is used to handle subscriber provisioning.
011 * <p>
012 * You can add subscribers in two ways:
013 * <ul>
014 * <li>Admin – can create (multiple) subscribers directly using the web GUI.
015 * <p>
016 * click the link: https://portal.kandy.io/my-kandy <br>
017 * and click on: Manage
018 * <li>Users – can create individual subscribers using the Reference
019 * Application:
020 * <ol>
021 * <li>The user opens the application and is prompted for a username, password,
022 * and phone number
023 * <li>The Kandy server sends an SMS to the supplied phone number, with a
024 * validation code
025 * <li>The user enters this code to complete the registration process
026 * </ol>
027 * </ul>
